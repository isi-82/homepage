RPC with Python

Create a application at the Discord Developer Portal
Customize it with a Picture etc.
Locate you'r Client ID and copy it
Save the application
Now open a Terminal and enter "pip install pyprecence"
Create a new Python file
Now import time and from pyprecence "Presence"
Create a variable called "client_id" with the value of your client id
Than initialize the Presence client with "RPC = Presence(client_id)"
Connect it to discord with "RPC.connect()"
Update your Presence with "RPC.update(status="text")"
and lastly create a wilhle true loop that runs the funciton "time.sleep(15)"
